## About the App / Uygulama Hakkında
İzimde is a personal emergency alert system that allows you to send instant SOS alerts to your emergency contacts with your live GPS location.

İzimde, acil durum kişilerinize anlık GPS konumunuzla SOS uyarısı göndermenizi sağlayan kişisel acil durum alarm sistemidir.
